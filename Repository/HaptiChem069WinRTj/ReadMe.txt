HaptiChem プログラムについて

------------------------------------------------------------
【準備】

この「ReadMe.txt」が入っているディレクトリ「HaptiChem069WinRTj」
をハードディスクの中の好きな場所へ丸ごとコピーします。

コピーしたものに対して、以下の操作を施してください。

------------------------------------------------------------
【起動】

HaptiChem プログラムを起動するには、「visual.bat」を
ダブルクリックします。
スクリーンの左上に HaptiChem のランチャーが出現します。

------------------------------------------------------------
【終了】

終了するには HaptiChem のランチャーを閉じます。「Really quit?」
または「本当に終了しますか？」という確認ダイアログが出ますの
で、「Yes」または「はい」のボタンを押します。
